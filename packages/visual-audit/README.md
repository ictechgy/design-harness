# @design-harness/visual-audit

Playwright-based screenshot capture and conservative DOM, layout, accessibility, and interaction checks for Design Harness.

Most users should start with the CLI:

```bash
npx @design-harness/cli@latest audit --url http://localhost:3000 --out runs/demo
```

Use this package directly when embedding the audit runner into another local tool.

Repository: https://github.com/ictechgy/design-harness
