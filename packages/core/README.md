# @design-harness/core

Core schemas, validation, scoring, report rendering, and shared types for Design Harness.

Most users should start with the CLI:

```bash
npx @design-harness/cli@latest audit --url http://localhost:3000 --out runs/demo
```

Use this package directly when building adapters or validating Design Harness artifacts in another tool.

Repository: https://github.com/ictechgy/design-harness
